# Windows版参照素材

三村囲碁オンラインは、元Windowsソフトの操作感と機能をブラウザへ移植するプロジェクトとして扱う。
このディレクトリは、CodexがWindows版の正解挙動を復元するための参照素材置き場。

## 置くもの

- `screenshots/`: Windows版の画面スクリーンショット
- `recordings/`: 通常操作の録画。無言録画でよい
- `samples/`: Windows版から出力したXML、SGF、設定ファイル、棋譜ファイル
- `notes/`: 操作メモ、観察ログ、不明点への回答

## 録画してほしい最小フロー

1. Windows版を起動する
2. 先生管理画面を開く
3. 教室を選んで開く
4. 生徒を入室させる、または入室済み状態を表示する
5. 対局を作る
6. 講義または共有検討を開く
7. 棋譜やXMLを読み込む操作があれば実行する
8. 授業を閉じる

解説音声は不要。画面・クリック・入力・ファイル選択結果が分かれば、Codex側で `tasks/windows-parity.md` に正解表を作る。

## 判断ルール

- Windows版の内部コード解析は原則不要。外から見た挙動、入出力ファイル、画面変化を正解とする。
- 不明点は「要Windows確認」として残し、三村さんには最後に業務判断だけ確認する。
- ブラウザ版で不要な機能は、削除ではなくまず「不要候補」として記録する。
