# 三村囲碁オンライン サービス戦略

作成日: 2026-06-30  
前提資料:

## 1. 結論

三村さんがやるべきサービスは、安いオンライン指導碁でも、単なるオンライン碁会所でもない。

**「三村九段に継続して見てもらえる、教室型オンライン囲碁サービス」**にする。

売るものはシステムではなく、以下の組み合わせ。

1. 三村智保九段の実績と人物像
2. 市川こども囲碁道場の育成実績
3. 一人ひとりを見て、次の課題まで出す継続指導
4. ブラウザだけで対局・検討・宿題・棋譜添削がつながる利便性

競合に対する立ち位置:

- 横浜囲碁サロンに対して: YouTube発信力では追う側。三村さんは「育成実績」と「教室運営の厚み」で勝つ。
- ダイヤモンド囲碁サロンに対して: 高級サロン感では戦わない。「安心して続けられる先生との関係」で勝つ。
- 洪道場に対して: 最強棋士輩出ブランドでは正面衝突しない。「本格的だが怖くない」「親が安心できる」「大人も入れる」で棲み分ける。

## 2. 商品の柱

### A. 大人向けオンラインサロン

目的: 低単価でも継続するストック収益を作る。サロン天元のオンライン版。

対象:

- 囲碁を続けたい大人
- 碁会所に行く心理的ハードルがある人
- 強くなりたいが、厳しい道場は怖い人
- 三村さんの解説・人柄に触れていたいファン

提供内容:

- 週1回のオンライン対局会
- 月1回の三村九段ミニ講座
- 月1回の公開棋譜診断、または抽選添削
- 会員向けコラム、棋戦解説、詰碁
- online-go-school 上での対局・検討

重要な制限:

- 個別添削を標準で付けすぎない。
- 「質問し放題」はやらない。質問は月1回募集、または講座内で回答。
- 三村さんの時間を食い潰さない設計にする。

### B. 子ども・級位者向けオンライン教室

目的: 現在のネット生を `online-go-school` に移し、ネット囲碁学園依存を減らす。

対象:

- 子ども
- 初段を目指す級位者
- 近くに強い子がいない地方の生徒
- 保護者が「本格的に見てほしい」と考えている家庭

提供内容:

- 週1回または複数回の集団対局
- 三村九段または講師による講評
- 詰碁宿題
- 棋譜保存と振り返り
- 必要に応じたKataGo分析
- 保護者向けの簡単な成長フィードバック

価値:

- Zoomと外部対局サイトを行き来しない。
- 先生が対局・検討・宿題を一続きで見られる。
- 保護者が「放置されていない」と感じられる。

### C. 棋譜添削・課題設計

目的: 先生の拘束時間をリアルタイムから非同期へ移し、収益性を上げる。

対象:

- 対局は自分で打てるが、どこを直せばよいか分からない人
- 大人の上達層
- オンライン教室の追加価値を求める家庭

提供内容:

- 生徒がSGFを提出
- 先生が重点局面を数箇所コメント
- 3〜5分の音声または短い動画で返却
- 次回までの課題を1つだけ出す

価値:

- 「見てもらえている」感覚が残る。
- リアルタイム指導より先生側の時間を調整しやすい。
- LP上で強い差別化になる。

### D. 少人数高単価コーチング

目的: 三村さんの時間を高単価で使う。最初から広げない。

対象:

- 本気で伸ばしたい子ども
- 保護者が教育投資として支払う家庭
- 短期集中で棋力を上げたい大人

提供内容:

- 月2回の個別面談または個別検討
- 月4局程度の棋譜添削
- 課題設計
- 月1回の保護者フィードバック
- 期間は3ヶ月単位

制限:

- 募集は3〜5名まで。
- 「24時間質問し放題」はやらない。
- 質問対応は週1回まとめて回答など、時間枠を明示する。

## 3. 価格設定

### 推奨価格

| 商品 | 価格 | 位置づけ |
|---|---:|---|
| 無料メール/YouTube | 0円 | 認知・信頼形成 |
| 単発体験会 | 1,100〜3,300円 | 参加ハードルを下げる。無料でもよいが、本気度を見たいなら有料 |
| 大人向けオンラインサロン | 3,980円/月 | ストック収益の入口 |
| 棋譜添削ライト | 5,500円/月 | 月2局程度。サロン会員向け追加商品 |
| 棋譜添削スタンダード | 11,000円/月 | 月4局程度。大人上達層向け |
| オンライン教室 週1 | 12,000円/月 | 現ネット生の基準価格 |
| オンライン教室 週2 | 22,000円/月 | 現行に近い複数回参加 |
| オンライン教室 週3 | 30,000円/月 | 本気層 |
| 少人数コーチング | 39,800〜49,800円/月 | 3〜5名限定。三村さん本人の時間を使う商品 |
| 3ヶ月集中パック | 99,000〜148,000円/3ヶ月 | 高単価商品の見せ方。月額より成果期間を売る |

### 価格思想

- 幽玄の間のような「安い単発指導碁」と比べられる場所に行かない。
- 価格の根拠は「対局1回」ではなく「継続して見てもらえること」。
- 低価格帯はサロンに限定し、個別対応は追加料金にする。
- 三村さん本人の時間が入るほど価格を上げる。

### 最初に出すべき現実的な3商品

1. **三村囲碁オンライン サロン**
   - 3,980円/月
   - 大人向け、週1対局会 + 月1講座

2. **オンライン教室 週1**
   - 12,000円/月
   - 子ども・級位者向け、現ネット生の移行先

3. **棋譜添削ライト**
   - 5,500円/月
   - サロンまたは教室の追加商品

高単価コーチングは、最初から大きく出さず、運用が回った後に限定募集する。

## 4. 顧客獲得の道筋

### 基本方針

広告費を先に使わない。三村さんの既存資産を順番につなぐ。

既存資産:

- 市川こども囲碁道場
- 囲碁サロン天元
- 志通信
- みむいご通信
- YouTube / Podcast
- 道場出身プロ・所属プロとの関係
- 保護者・支援者コミュニティ
- online-go-school

### 獲得導線

1. **認知**
   - YouTubeで棋戦解説、上達講座、オンライン教室の様子を発信。
   - みむいご通信で一般囲碁ファンを集める。
   - サロン天元・道場サイトにオンライン導線を置く。

2. **信頼形成**
   - 三村九段の指導観を書く。
   - 道場の育成実績を出す。
   - 「練習量は裏切らない」「次の課題まで見る」を繰り返し伝える。
   - 実際のレッスン風景、講評例、添削例を見せる。

3. **低ハードル参加**
   - 月1回のオンライン公開対局会。
   - 体験レッスン。
   - 棋譜診断ライブ。
   - 無料または低価格のメール講座。

4. **月額化**
   - 大人はオンラインサロンへ。
   - 子どもはオンライン教室へ。
   - 上達意欲が強い人には棋譜添削を追加。

5. **高単価化**
   - 継続会員の中から、少人数コーチングへ案内。
   - 最初から売り込まず、3ヶ月単位で限定募集する。

## 5. セグメント別の訴求

### 子ども・保護者

刺さる訴求:

- プロ棋士が継続して見る。
- 何を勉強すればよいかが分かる。
- 家にいても強い相手と対局できる。
- 保護者が成長を把握できる。

避ける訴求:

- 「楽しいだけ」
- 「誰でもすぐ強くなる」
- 「安い」

### 大人・シニア

刺さる訴求:

- 恥をかかずに参加できる。
- 怖くない。
- 三村九段の解説が聞ける。
- 碁会所に行かなくても仲間ができる。
- 自分の対局を見てもらえる。

避ける訴求:

- 競争を煽りすぎる。
- プロ志望向けの厳しさを前面に出しすぎる。
- システム機能だけを説明する。

### 本気層

刺さる訴求:

- 課題を明確にする。
- 棋譜を継続的に見る。
- 努力量と方向性を管理する。
- 市川道場の育成実績。

避ける訴求:

- 安易な成果保証。
- 入段保証。
- 三村さんの無制限対応。

1. ヒーロー
   - 「三村九段が、一人ひとりの対局と成長を見るオンライン囲碁教室」
   - システム名より、先生と関係性を前面に出す。

2. こんな人へ
   - 近くに囲碁教室がない
   - 自分の碁を見てほしい
   - 子どもに本格的な指導を受けさせたい
   - 碁会所は少し怖いが、囲碁仲間がほしい

3. 三村智保九段について

   - 市川こども囲碁道場主宰
   - 育成実績
   - 指導観

4. サービス内容
   - オンラインサロン
   - オンライン教室
   - 棋譜添削

5. 使い方
   - ブラウザで入室
   - 対局
   - 検討
   - 宿題・棋譜添削

6. 料金
   - 3商品に絞る

7. 体験参加
   - 申込フォーム
   - 初回体験会の日程

## 7. 最初の90日

### 0〜30日

- 既存生徒1〜3人で試験レッスン。
- 音声・入室・対局・検討の詰まりを潰す。
- 実際の講評例を1〜2本作る。
- LPの仮版を作る。

### 31〜60日

- 大人向けオンライン対局会を月1回テスト開催。
- サロン天元の常連、YouTube、みむいご通信で案内。
- 3,980円/月のオンラインサロンを小さく募集。
- 定員は最初10〜20名でよい。

### 61〜90日

- 棋譜添削ライトを追加。
- オンライン教室週1の正式募集。
- 体験会から月額への転換率を確認。
- 高単価コーチング候補を既存会員から探す。

## 8. 成功指標

最初に見る数字:

- 体験会参加者数
- 体験後の月額転換率
- サロン継続率
- 棋譜添削の消化時間
- 先生側の週あたり拘束時間
- 入室トラブル件数
- 保護者・大人会員からの定性フィードバック

売上より先に見るべきこと:

- 三村さんの時間が破綻していないか。
- 参加者が「見てもらえている」と感じているか。
- システムが関係性を邪魔していないか。

## 9. やらないこと

- 安い単発指導碁の大量販売。
- 質問し放題。
- 入段保証、成果保証。
- 最初から大規模広告。
- 機能一覧だけのLP。
- 三村さん本人の時間を無制限に使う商品。

## 10. 最重要メッセージ

三村囲碁オンラインは、便利な対局システムではない。

三村九段が、一人ひとりの碁を見て、次の課題まで一緒に整えるオンラインの教室。

この言い方を軸に、商品・価格・LP・発信を揃える。

## 11. ネット囲碁学園から自前アプリへ移る意味

現在のネット囲碁学園は、Windowsデスクトップアプリとして既存のオンライン授業を支えている重要な基盤。ただし、教室利用には本来有料課金が必要であり、開発協力者かつ知人である方から無料提供を受けている状態にある。

この状態には、金銭面だけでなく心理的・事業的な制約がある。

### 自前アプリ化の本質的メリット

1. **心理的負担から解放される**
   - 知人の好意に依存して事業を続ける状態から抜けられる。
   - 顧客から月謝を受け取るサービスで、基盤だけ無償提供を受け続ける心苦しさを減らせる。
   - 相手への感謝は残しつつ、事業基盤としては自立できる。

2. **事業上の独立性が上がる**
   - 価格、機能、導線、顧客管理を自分の判断で変えられる。
   - 新規顧客向けに「このアプリを入れてください」ではなく「このURLから参加してください」にできる。
   - ネット囲碁学園側の仕様・課金・対応OS・更新方針に左右されにくい。

3. **料金受け取りをアプリ内に統合できる**
   - 体験会申込、月額サロン、オンライン教室、棋譜添削、個別コーチングを同じ導線で販売できる。
   - Stripe等と連携すれば、申込から決済、参加権限、レッスン入室までを一体化できる。
   - 「支払い済みの人だけ入室」「プランごとに機能解放」「継続課金の管理」が可能になる。

4. **顧客データと学習履歴を蓄積できる**
   - 棋譜、講評、宿題、出席、添削、決済状況を同じ顧客に紐づけられる。
   - 保護者や大人会員に「継続して見てもらっている」体験を作れる。
   - 将来のサロン、添削、教材販売へ展開しやすい。

5. **ブランドを自前化できる**
   - 顧客から見る入口を「ネット囲碁学園」ではなく「三村囲碁オンライン」にできる。
   - システム名ではなく、三村九段の教室・人物像・指導観を前面に出せる。
   - 競合との差別化が、外部ツール利用ではなく自前の教育体験になる。

### 注意点

- ネット囲碁学園は既存授業を支えてきた実績があるため、すぐ全面移行しない。
- 既存ネット生には、慣れた環境を急に変える負担がある。
- 自前アプリは、入室、音声、対局、検討、講評まで実レッスンで安定確認が必要。
- 決済を入れる前に、参加権限・プラン管理・返金対応・休会対応の運用を決める。

### 推奨ステップ

1. 既存ネット生1〜3人で試験レッスン。
2. 新規体験会は最初から三村囲碁オンラインで実施。
3. 問題なければ、オンラインサロンと棋譜添削を三村囲碁オンライン側に載せる。
4. Stripe等で体験会・月額・添削の決済を統合。
5. 既存ネット生は段階的に移行し、ネット囲碁学園は当面バックアップとして残す。

結論:

三村囲碁オンラインを作る意味は、単なる機能改善ではない。  
**好意に依存している現在の基盤から自立し、顧客獲得・料金受け取り・学習履歴・ブランドを自分の事業資産として持つこと**にある。
